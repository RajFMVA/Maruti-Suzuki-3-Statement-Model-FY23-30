# Maruti Suzuki – 3-Statement Financial Model

## Overview
This repository contains a fully integrated 3-statement financial model for Maruti Suzuki India Ltd, built using audited standalone annual reports.

The model forecasts the Income Statement, Balance Sheet, and Cash Flow Statement with internally consistent linkages and no reliance on screening platforms.

## Key Features
- Integrated IS–BS–CF model with zero balance sheet breaks
- Driver-based forecasting assumptions
- Clean cash flow reconciliation
- Conservative balance sheet modeling
- Historical and forecast financial visualization

## Data Source
- Maruti Suzuki India Ltd – Annual Reports (Standalone Financials)

## Tools Used
- Microsoft Excel
